📦 Deep Dive — Large File Chunking Strategy

When a file exceeds 500 lines or 1 MB, it is split into overlapping windows before being sent to the LLM. The 50-line overlap ensures that context at chunk boundaries is never lost — a smell spanning two chunks will still appear in full within one of them.

Example: 600-line file → 2 chunks
Chunk 1 Lines 1 → 500
↑ 50-line overlap
Chunk 2 Lines 451 → 600
Lines 451–500 appear in both chunks — boundary smells are never truncated.
📐
Chunk Parameters
Max lines/chunk: 500
Overlap: 50 lines
Max tokens/chunk: 3,000
🔄
Per-Chunk Flow
1. Slice lines[start:end]
2. Build detection prompt
3. Send to LLM
4. Parse JSON response
5. Adjust line numbers (+start)
🔗
Merge Strategy
All chunk results appended to all_smells[]. Smell IDs use MD5 of path+line+type — no duplicates. Final sort by severity + confidence.
